# Code

## Notebooks
Python Jupyter notebook (.ipynb) file. Files should be prefixed with the order in which they run:
00_file1.ipynb
01_file2.ipynb
02_file3.ipynb

### Scripts
Python script (.py). Scripts should be prefixed with the order in which they run:
00_script1.ipynb
01_script2.ipynb
02_script3.ipynb
